# Data-Science-and-Artificial-Intelligence

Plaksha University – Course FM211
Introduction to Artificial Intelligence – Lab Work

This repository contains the coursework and lab work completed as part of the Introduction to Artificial Intelligence (FM211) course at Plaksha University (Monsoon 2025–26) .

The work covers key topics in data science and AI, including data preprocessing, visualization, hypothesis testing, regression (linear, multiple, logistic), and decision trees. It also explores core AI concepts such as intelligent agents, search algorithms (BFS, DFS, A*), optimization, probabilistic reasoning, and basic machine learning techniques.
